# disc-buddy UI

Flutter desktop GUI for disc-buddy. Wraps the `disc_buddy` library with a graphical interface for ripping Audio CDs, DVDs, and Blu-rays on Linux.

See the [root README](../README.md) for full documentation, requirements, and output format.

## Build

```bash
flutter pub get
flutter build linux --release
# Output: build/linux/x64/release/bundle/
```

## Run from source

```bash
flutter pub get
flutter run -d linux
```
